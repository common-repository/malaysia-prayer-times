Widget that shows current or all prayer times for locations in Malaysia. Data by JAKIM.

== Description ==
This plugin add 2 types of widget showing current prayer time and all prayer times, for selected locations in Malaysia. The prayer times data is from [JAKIM e-Solat](http://www.e-solat.gov.my/) courtesy of jQuery API by [i906](http://mpt.i906.my). **Made into WordPress plugin by [Envigeek Web Services](http://www.envigeek.com/ "Envigeek Web Services")**.

Features:

* Each widget can be uniqued to specific location as selected
* Easy location search using Select2 search-as-typing dropdown
* Support multiple widgets per page
* Current prayer times highlighted with bootstrap-compliant .active class
* Show all prayer times can be shown in table list or inline list
* Customized text to appear for current prayer time, for each widget

For any plugin issues, bug reports or feature request, please submit your topic on [WordPress plugin support forum](http://wordpress.org/support/plugin/malaysia-prayer-times)

Banner photo on WP.org plugin page, credit to [Sulaiman](http://500px.com/photo/23527419)

== Installation ==
1. Upload and extract "malaysia-prayer-times.zip" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
1. Add the widgets to your sidebar. Choose location as required.

== Frequently Asked Questions ==
= Can I have multiple widgets per page? =
Yes, the plugin supports multiple widgets on the same page. Each widget can have its own locations.

= How to style the output? =
Use .wp-mpt class to style the table (for all prayer times) or div (current prayer time) element.
